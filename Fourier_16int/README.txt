Интерполяция Фурье для моно/стерео int16 сигналов.
Используется DCT(дискретное косинусное преобразование), т.к. по формуле из учебника получается комплексный сигнал.
Для сборки проекта необходимо установить пакетный менеджер MSYS2 для MINGW64, там прописать команду pacman -S mingw-w64-x86_64-fftw.
Если по каким-либо причинам проект сразу же закрывается, даже не успев открыться, необходимо вручную запустить собранный exeшник и посмотреть, каких DLL-ек ему не хватает
